Place one background image in this folder.

Supported file types:
- .png
- .jpg
- .jpeg
- .webp
- .avif
- .gif

How it works:
- The portfolio automatically uses the first image in this folder in alphabetical order.
- If you keep only one image here, that image becomes the website background.

Tip:
- After adding a new image, refresh the page.
- If the dev server does not pick up the new file immediately, restart `npm run dev`.
